# proyectoISSBC
Trabajo de valoración de la asignatura ISSBC


Correcciones en el software:

	-ckModValoracion: el sistema valora el problema pudiendo tomar en cuenta todos los criterios, y no solo aquellos hasta que se alcanza la puntuación mínima
	-domITV: incluye criterios con mayor precisión
	-esqConocimiento: eliminada la opción múltiple e incluidas dos nuevas opciones (opciones y varios) en la que el primero toma una lista de valores estáticos para ser repetida y el segundo permite incluir valores dinámicos
	-ckVtsValoracion: la descripcion de criterios devuelta como lista se ha convertido en una tabla de criterios descritos ordenados, se ha corregido la justificacion devuelta
